# COVID-19 Q/A Chatbot (Machine Learning + NLP + Tkinter GUI)

A machine-learning powered chatbot built using **TensorFlow**, **Keras**, and **NLTK**.  
It classifies user queries using NLP and responds intelligently through a custom-trained neural network model.  
The project includes a clean and interactive **Tkinter GUI** for desktop usage.

---

## ✨ Features

- 🧠 **Neural Network Model** (TensorFlow/Keras)
- 🔤 **NLP Processing** using NLTK (tokenizing, lemmatizing)
- 💬 Intent-based conversational responses
- 🪟 Clean Tkinter GUI interface
- 🎯 97% model accuracy after 200 training epochs
- ⚡ Real-time predictions
- 📁 JSON-based dataset for easy modification

---

## 🛠 Tech Stack

- **Python 3.7**
- **TensorFlow 2.6**
- **Keras**
- **NLTK**
- **NumPy**
- **Tkinter GUI**
- **JSON (intents dataset)**

---

## 🚀 How to Run

```bash
# 1. Create a Python 3.7 venv
py -3.7 -m venv venv
venv\Scripts\activate

# 2. Install dependencies
pip install -r requirements.txt

# 3. Train the model (optional)
python train_chatbot.py

# 4. Run the GUI application
python gui.py

```
